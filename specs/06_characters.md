## 脇役：王国審問官（CHAR-005）

| 項目 | 内容 |
|---|---|
| 名前 | ドルラン |
| 年齢 | 42歳 |
| 役割 | 王国法廷 主任審問官。第三幕の裁判を主導する「困惑役」 |
| 一人称 | 私（男性・法廷口調の無機質な「私」） |
| 口調 | 法廷用語・文語調。冷静・威圧的。困惑すると短文に崩れる |
| 機能 | 「悪行」を証拠提示するほど善行が明らかになり、収拾がつかなくなるコメディのツッコミ役 |
| 弧 | 自信満々で開廷→証言ごとに困惑が蓄積→法廷が騒乱になり締めくくれない |
| 差別化 | クロエ「私（女性・毒舌・感情あり）」と一人称は同じだが、男性・法廷専用・感情なしで声を区別 |

## 脇役：筆頭家令（CHAR-006）

| 項目 | 内容 |
|---|---|
| 名前 | アルダ |
| 年齢 | 52歳 |
| 役割 | ヴァルブレイズ家の筆頭家令（執事長）。領地運営・実務全般を仕切る |
| 一人称 | わたくしめ（老齢男性の謙譲表現） |
| 口調 | 穏やかで格調ある老練な家令口調。急かさず包み込む |
| 機能 | ガレットの「悪の指示」を忠実に実行し、すべて善行に変換して報告する「黒子の善行変換装置」。裁判でガレットの自爆を防ぐフォロー役 |
| 弧 | 穏やかな重臣→裁判でガレットの全自爆をフォロー→にぎやかな幕の一員 |
| 差別化 | クロエ（若い女性・即座の毒舌・日常密着）との対比: アルダは年配男性・鷹揚・実務・包み込む温かさ。エレオノーラ「わたくし」と「わたくしめ」で区別 |

|---|---|
