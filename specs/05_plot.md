# Plot Spec

## 構成

### 第一幕：裁判直前

- 召喚状が届いてから裁判の朝まで
- ガレットは「ついに俺の悪が認められる！」と意気揚々
- 主人公のキャラクターと世界観を確立
- 「なぜこんな裁判になったのか」という問いを置く（第二幕への導線）

### 第二幕：過去編（第一幕に至るまで）

- 主人公が悪を目指してきた日々のエピソード集
- 「悪いことをしようとして善行になる」パターンを複数積み重ねる
- 他者視点の挿入で「みんな知っている」が蓄積される
- 第三幕の裁判証拠がここで出揃う

### 第三幕：裁判

- 第二幕で描いた出来事が証拠として登場する
- 王国側が「悪行」を提示 → 当事者が感謝を述べる
- ガレットが自ら暴露しようとする → 側近・民衆がフォローして真実を歪める
- この二つが混在しながら展開し、収拾がつかなくなる
- ガレット、「バレていた」と気づく
- 「俺は悪だ！」と叫ぶが誰も信じない
- にぎやかに幕

## 第二幕エピソードの基本パターン

```
悪いことをしようとする
  → 実行できない / 形が変わる
    → 結果として善行になる
      → 周囲は「また良いことした」と思う
        → ガレットは「うまくいかなかった」と落ち込む
```
