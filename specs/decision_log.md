# Decision Log

## DECISION-001

- Phase: Phase 0 — Project Kickoff
- Decision: 文字数 5〜8万字、読後感はカタルシス（誤解が解けた爽快感）、舞台は純粋な異世界ファンタジー、勘違いの軸は「主人公が悪を演じようとするが周囲にはバレている」
- Options Considered: 短編(2〜4万字) / 標準中編(5〜8万字) / 長め中編(8〜12万字)、読後感A〜D、ジャンルA〜F
- Reason: 標準的な中編はエピソード構造と裁判の両立に適切な長さ。カタルシスはコメディとの相性が良い。
- Impact: 全体の規模感・トーン・構造を規定する
- Status: approved

---

## DECISION-002

- Phase: Phase 1 — Concept Spec
- Decision: 主人公は悪名を持つ貴族・領主。動機は一貫して「悪への純粋な憧れ」（変化なし）。悪行が善行になる。中心事件は王国の審問・裁判。カタルシスは「裁判で善行が証明される＋誰も悪を信じない」。
- Options Considered: 動機A（守るため）/ B（改心後）/ C（目的のため）/ D（純粋な憧れ）、「悪名が盾」要素は検討後に除外
- Reason: 「悪名が盾」を入れると感情の山が分散するため、「バレていたと気づく瞬間」に一本化した。
- Impact: 物語全体の構造と感情曲線の中心軸
- Status: approved

---

## DECISION-003

- Phase: Phase 2 — Reader Experience Spec
- Decision: 読者は主人公の内面と周囲がバレている様子の両方を知っている。他者視点を時々挿入。主人公への感情は愛おしさ＋もどかしさ。全編コメディベース、感動は隠し味。
- Options Considered: 視点A（主人公のみ）/ B（両方知っている）/ C（複数視点交互）
- Reason: 読者が全部知っている状態がコメディを強化する。「早く気づいてくれ！」のもどかしさが感動への助走になる。
- Impact: 文体・視点設計・感情曲線
- Status: approved

---

## DECISION-004

- Phase: Phase 4 — Ending Spec
- Decision: 裁判でガレットが「俺は悪だ」と主張し続けるが誰にも信じてもらえず終わる。主人公の変化なし。ラストはにぎやかで明るい。一人称は「俺」。
- Options Considered: 裁判結末A（無罪放免）/ B（有罪だが裁けない）/ C（主張するが信じてもらえない）/ D（裁判を超えた何か）
- Reason: 主人公が最後まで一貫していることで笑いと愛おしさが両立する。「信じてもらえない」が否定ではなく肯定として届く。
- Impact: 結末の設計・カタルシスの形・ラストシーンの質感
- Status: approved

## DECISION-005

- Phase: Phase 5 — Plot Spec
- Decision: 三幕構成。第一幕は裁判直前、第二幕は過去編（エピソード集）、第三幕は裁判。非線形構造。
- Options Considered: 起点A（召喚状から）/ B（日常から）/ C（新キャラ登場から）、中盤A（エピソード集）/ B（証拠集め）/ C（本物の悪事を試みる）
- Reason: 第二幕を過去編にすることで、読者が「この善行がどう裁判に繋がるか」を知りながら読める。伏線回収の快感が強まる。
- Impact: 物語の時系列構造・読者の期待設計
- Status: approved

## DECISION-006

- Phase: Phase 9 — Scene Spec（部分）

## DECISION-007

- Phase: Phase 6 — Character Spec（ヒロイン3人）
- Decision: 3人とも「ヤンデレ」だが出方を差別化。クロエ(メイド/15)=毒舌の仮面で隠す崇拝、ヴィオラ(同級生/16)=勝気な独占宣言型、エレオノーラ(王女/16)=完璧な仮面で権力行使型。エレオノーラは救出時にガレットの本音の独り言を聞いており、その証言が終盤の切り札となる。
- Options Considered: 各ヒロインの接し方・属性・裁判での役割・ヤンデレの出方
- Reason: 全員ヤンデレでも「隠す/宣言/権力」と出方を分けることで被りを回避。三つ巴の修羅場がにぎやかなラスト(DECISION-004)に直結する。
- Impact: bible/characters.yaml(CHAR-002〜004)、第二幕エピソードの担い手、裁判の証言構造、ラストの修羅場
- Status: approved

## DECISION-008

- Phase: Phase 6 — Character Spec（ヒロイン口調確定）
- Decision: 一人称と主人公の呼び方を確定。クロエ=「私」/お館様,あなた、ヴィオラ=「アタシ」/アイツ(三人称),アンタ(本人)、エレオノーラ=「わたくし」/ガレット,あなた。ヴィオラは苛烈・伝法な口調(気高い令嬢なのに口が悪いギャップ)。崩れ方も3人で差別化(静かに重い/自己否定/気品のまま暴走)。
- Options Considered: ヴィオラ一人称「わたくし」案 vs 「アタシ」案
- Reason: 一人称が「私/アタシ/わたくし」で綺麗に割れ、書き始めてからの声のブレを防ぐ。共通場面のセリフサンプルで検証済み。
- Impact: bible/characters.yaml の各 speech セクション、本文の会話文すべて
- Status: approved

## DECISION-009

- Phase: Phase 9/10 — 冒頭試し書き（文体・視点フォーマット確定）
- Decision: 文体は一人称(ガレット視点)・地の文やや硬め・コメディは会話と取り繕いで効かせる。S01(manuscript/S01.md)を冒頭の基準とする。他者視点の挿入は「◇」で区切る形式を標準フォーマットとする。重要: ガレットが「ずっとバレていた」と気づくのは終盤(裁判)のため、それ以前のシーンでガレット本人に真相を悟らせない。ヒロインの本音は◇以降の他者視点で読者にのみ開示する。
- Options Considered: クロエの本音をガレットに聞かせる案(却下) / 読者にだけ届ける◇視点切替案(採用) / セリフ自体を削る案
- Reason: Ending Spec(気づきは裁判で)とReader Experience Spec(読者>主人公の情報差)を両立させる。冒頭からもどかしさのフックを立てつつ、終盤の山を温存する。
- Impact: 全シーンの視点運用、第二幕の他者視点挿入(クロエ/ヴィオラ/エレオノーラ)、manuscript全体の文体基準
- Status: approved

## DECISION-010

- Phase: Phase 7 — World Spec + 脇役（サブエージェント設計を承認）
- Decision: 国名=アルデニア王国、王都=テルハイン。辺境の自治権・王国法廷・貴族学院を設定。魔法は中立で「悪行→善行」はガレットの人格の問題(魔法のせいにしない)。法廷は証言者の主観評価も証拠になりうる(裁判コメディの土台)。脇役: ドルラン(CHAR-005/審問官/困惑役)、アルダ(CHAR-006/筆頭家令/善行変換の黒子)、イレーヌ(CHAR-007/孤児院シスター/感謝の証言者)。孤児院は裁判の証言のみで第二幕に独立エピソードは作らない。
- Options Considered: 王国名・脇役の役職・イレーヌの要否・孤児院を第二幕に出すか
- Reason: ヴァルブレイズの語感に整合。脇役の一人称・役割を既存5人と差別化。設定を作り込みすぎず中編コメディに必要な最小限に留める。
- Impact: specs/07_world.md, bible/world.md・rules.yaml・locations.yaml・timeline.yaml、bible/characters.yaml(CHAR-005〜007)、specs/06_characters.md
- Status: approved

## DECISION-011

- Phase: Phase 9 — Scene Spec（第二幕の各ヒロイン2話化）
- Decision: 第二幕を各ヒロイン2話に拡張(全8話→全11話)。各エピソードは2パート構成(IDはS06-08を動かさぬよう接尾辞分割)。クロエ=S03-1悪の経営(コメディ)/S03-2拾われた日(掘り下げ)。ヴィオラ=S04-1学園(公の対立・設定)/S04-2尾行(素の目撃・恋落ち)※公の顔→素顔の順。エレオノーラ=S05-1救出(事件)/S05-2切り札(掘り下げ)。S05の事件は「魔物の襲撃」、王女は辺境視察中に襲われガレットが救う。2話目は悪行→善行ギャグの反復にせず掘り下げ・関係性・伏線に役割を振る(単調回避)。
- Options Considered: 各ヒロイン1話vs2話、S04の話順(尾行先vs学園先→学園先採用)、S05の事件種別(魔物A/災害B/野盗C→A採用)
- Reason: 第二幕=各ヒロインの掘り下げが本作の心臓部。2話化で深みを出しつつ、役割を分けてギャグの連打による飽きを防ぐ。
- Impact: plot/scenes.yaml(全11シーン)、plot/acts.yaml、第二幕の執筆量・全体尺(5〜8万字)
- Status: approved

## DECISION-012

- Phase: Phase 10 — 尺の調整方針
- Decision: 目標5〜8万字に対し現ペース(1話約1,500字)では不足するため、(1)本編は1話平均4,000〜5,000字を目安に厚く書く、(2)閑話(本筋外の小エピソード)を3〜4本挿入する、の併用で調整する。閑話は「日常の悪行未遂」「学院のドタバタ」「裁判前夜の集結」等とし、裁判の証拠(善行)を増やす役割も兼ねる。S01〜S03-1は骨組み寄りのため、後で肉付けの対象とする。執筆と並行して累計字数を追跡する。
- Options Considered: 本編を厚くするのみ / 閑話で稼ぐのみ / 両方併用(採用)
- Reason: 本編だけ無理に伸ばすと冗長化・質低下のリスク。閑話だけだと本筋が薄いまま。両方で質を保ちつつ尺を確保する。
- Impact: 全本編の長さ、閑話の新規追加、plot/scenes.yaml、全体尺
- Status: approved
- 進捗(随時更新): S01=1423 / S02=1108 / S03-1=1777 / S03-2=2136 ／ 累計 約6,400字（残り7話＋閑話3本で約45,000〜50,000字積む想定）
