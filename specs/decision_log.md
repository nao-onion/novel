# Decision Log

## DECISION-001

- Phase: Phase 0 — Project Kickoff
- Decision: 文字数 5〜8万字、読後感はカタルシス（誤解が解けた爽快感）、舞台は純粋な異世界ファンタジー、勘違いの軸は「主人公が悪を演じようとするが周囲にはバレている」
- Options Considered: 短編(2〜4万字) / 標準中編(5〜8万字) / 長め中編(8〜12万字)、読後感A〜D、ジャンルA〜F
- Reason: 標準的な中編はエピソード構造と裁判の両立に適切な長さ。カタルシスはコメディとの相性が良い。
- Impact: 全体の規模感・トーン・構造を規定する
- Status: approved

---

## DECISION-002

- Phase: Phase 1 — Concept Spec
- Decision: 主人公は悪名を持つ貴族・領主。動機は一貫して「悪への純粋な憧れ」（変化なし）。悪行が善行になる。中心事件は王国の審問・裁判。カタルシスは「裁判で善行が証明される＋誰も悪を信じない」。
- Options Considered: 動機A（守るため）/ B（改心後）/ C（目的のため）/ D（純粋な憧れ）、「悪名が盾」要素は検討後に除外
- Reason: 「悪名が盾」を入れると感情の山が分散するため、「バレていたと気づく瞬間」に一本化した。
- Impact: 物語全体の構造と感情曲線の中心軸
- Status: approved

## DECISION-003

- Phase: Phase 2 — Reader Experience Spec
- Decision: 読者は主人公の内面と周囲がバレている様子の両方を知っている。他者視点を時々挿入。主人公への感情は愛おしさ＋もどかしさ。全編コメディベース、感動は隠し味。
- Options Considered: 視点A（主人公のみ）/ B（両方知っている）/ C（複数視点交互）
- Reason: 読者が全部知っている状態がコメディを強化する。「早く気づいてくれ！」のもどかしさが感動への助走になる。
- Impact: 文体・視点設計・感情曲線
- Status: approved

## DECISION-004

- Phase: Phase 4 — Ending Spec
- Decision: 裁判でガレットが「俺は悪だ」と主張し続けるが誰にも信じてもらえず終わる。主人公の変化なし。ラストはにぎやかで明るい。一人称は「俺」。
- Options Considered: 裁判結末A（無罪放免）/ B（有罪だが裁けない）/ C（主張するが信じてもらえない）/ D（裁判を超えた何か）
- Reason: 主人公が最後まで一貫していることで笑いと愛おしさが両立する。「信じてもらえない」が否定ではなく肯定として届く。
- Impact: 結末の設計・カタルシスの形・ラストシーンの質感
- Status: approved

## DECISION-005

- Phase: Phase 5 — Plot Spec
- Decision: 三幕構成。第一幕は裁判直前、第二幕は過去編（エピソード集）、第三幕は裁判。非線形構造。
- Options Considered: 起点A（召喚状から）/ B（日常から）/ C（新キャラ登場から）、中盤A（エピソード集）/ B（証拠集め）/ C（本物の悪事を試みる）
- Reason: 第二幕を過去編にすることで、読者が「この善行がどう裁判に繋がるか」を知りながら読める。伏線回収の快感が強まる。
- Impact: 物語の時系列構造・読者の期待設計
- Status: approved

## DECISION-009

- Phase: Phase 9/10 — 冒頭試し書き（文体・視点フォーマット確定）
- Decision: 文体は一人称(ガレット視点)・地の文やや硬め・コメディは会話と取り繕いで効かせる。S01(manuscript/S01.md)を冒頭の基準とする。他者視点の挿入は「◇」で区切る形式を標準フォーマットとする。重要: ガレットが「ずっとバレていた」と気づくのは終盤(裁判)のため、それ以前のシーンでガレット本人に真相を悟らせない。ヒロインの本音は◇以降の他者視点で読者にのみ開示する。
- Options Considered: クロエの本音をガレットに聞かせる案(却下) / 読者にだけ届ける◇視点切替案(採用) / セリフ自体を削る案
- Reason: Ending Spec(気づきは裁判で)とReader Experience Spec(読者>主人公の情報差)を両立させる。冒頭からもどかしさのフックを立てつつ、終盤の山を温存する。
- Impact: 全シーンの視点運用、第二幕の他者視点挿入(クロエ/ヴィオラ/エレオノーラ)、manuscript全体の文体基準
- Status: approved

## DECISION-012

- Phase: Phase 10 — 尺の調整方針
- Decision: 目標5〜8万字に対し現ペース(1話約1,500字)では不足するため、(1)本編は1話平均4,000〜5,000字を目安に厚く書く、(2)閑話(本筋外の小エピソード)を3〜4本挿入する、の併用で調整する。閑話は「日常の悪行未遂」「学院のドタバタ」「裁判前夜の集結」等とし、裁判の証拠(善行)を増やす役割も兼ねる。S01〜S03-1は骨組み寄りのため、後で肉付けの対象とする。執筆と並行して累計字数を追跡する。
- Options Considered: 本編を厚くするのみ / 閑話で稼ぐのみ / 両方併用(採用)
- Reason: 本編だけ無理に伸ばすと冗長化・質低下のリスク。閑話だけだと本筋が薄いまま。両方で質を保ちつつ尺を確保する。
- Impact: 全本編の長さ、閑話の新規追加、plot/scenes.yaml、全体尺
- Status: approved
- 進捗(随時更新): S01=2193/S02=1430/S03-1=1936/S03-2=2136/S04-1=2488/S04-2=2109/S05-1=2770/S05-2=2012 ／ 累計 約17,100字(8/13)。残りSAN-1/SAN-2/S06/S07/S08の5シーン。S07/SANの長尺化(各6-8k)とB(既存肉付け)で約33,000字積む必要。
- 補足: 裁判の発端=隣領バルゴア伯爵の妬みによる審問奏上(S05-2)。ガレット自身の悪名アピールが証拠になり潔白証明を難しくしている、という皮肉を裁判で使う。

## DECISION-015

- Phase: Phase 11/12 — 校閲(重複表現)＋エレオノーラ独白の方針(Alt A)
- Decision: (1)サブエージェント校閲で検出した重複を修正。最重要「その『いつか』が〜まだ知らない」型の締めが4話で反復→各話を別の締めに差別化(S03-1イレーヌ/S04-1/S04-2は語句変更、S05-1は『知る由もない』に1本化)。S01とS03-1の独白ほぼ同文(飼い殺し/骨の髄/俺なしでは生きられぬ/蜘蛛/甘言)を語彙分散。凪いだ(S01×3)・不敵(S01×2)・口角/唇の端(3話)・手駒(S04-1/S05-1)・胸の奥(S04-1/S05-1)等を分散。
- (2)エレオノーラの救出独白はAlt A採用: 個人愛(「お前だけは/お前だから」)をやめ、普遍的な無私の信条(「誰ひとり死なせない/王女だろうが民だろうが関係ない/生きてくれ」)に変更。幼少期の特別な縁は作らない。エレオノーラは『特別だから』ではなく彼の本質的善良さを目撃して惚れる(ハーレム均衡維持＋S08切り札の強化)。S01の大魔王の夢設定は変更せず。
- Options Considered: 幼少期縁①(却下:重心偏り＋魔王の夢の約束が構造と不整合)/②/Alt A(採用)/B/C
- Reason: ユーザー選択。独白の飛躍感の原因は『親密さの飛躍』→普遍的信条にすれば飛躍が消え、過重もなく切り札も強い。重複は読者の既視感リスクのため校閲指摘を反映。
- Impact: manuscript S01/S03-1/S04-1/S04-2/S05-1、bible/characters.yaml(CHAR-004 secret/scene refs)
- Status: approved
- 文体校閲レポートは subagent 実施(検出のみ)、修正は手動。残課題: S02等の軽微な底上げ(B)、今後の各話でも締めパターンの反復に注意。

## DECISION-013

- Phase: Phase 10 — 文体方針
- Decision: 地の文はやや軽め・語彙はやさしめにする。難しい熟語(例: 泰然/忌み嫌われ/凡庸/値踏み/債権者 等の硬い表現)は避け、平易な言い回しと短めの文を基本とする。ただしガレットの悪ぶった台詞の大仰さ(キャラ由来)は維持する。既存のS01〜S03-2も今後この方針で微調整・肉付けする。
- Reason: ユーザーフィードバック「もう少しだけ文体軽めで、語彙も難しくなくていい」。読みやすさとコメディのテンポを優先する。
- Impact: 全本文の地の文、今後の執筆・肉付け全般
- Status: approved
- 追記(重要な例外): 軽め・やさしめは ◇他者視点(クロエ/アルダ/ヴィオラ/エレオノーラ視点)や説明的地の文に限定する。ガレットの台詞および一人称地の文の「大仰な語彙」はキャラの魅力なので維持する(ユーザー指摘「大仰な感じがよく出てた」)。

## DECISION-014
