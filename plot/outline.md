# Plot Outline

## 作品概要

悪に憧れる16歳の領主・ガレットが、何をやっても善行になってしまう。王国の審問・裁判で、積み重なった「善行」が次々と暴かれ、誰にも悪を信じてもらえないまま幕が閉じる。

## 三幕構成

```
第一幕（裁判直前）
  └─ 召喚状が届く → ガレット大喜び → 裁判の朝へ

第二幕（過去編）
  ├─ エピソード1：重税 → 感謝される（メイド）
  ├─ エピソード2：尾行されて素がバレる（同級生）
  └─ エピソード3：命がけ救出 → 気絶していなかった（王女）

第三幕（裁判）
  ├─ 王国側「悪行」提示 → 当事者が感謝
  ├─ ガレット自爆しようとする → 側近がフォロー
  ├─ 収拾がつかなくなる
  ├─ ガレット「バレていた」と気づく
  └─ 「俺は悪だ！」→ 誰も信じない → にぎやかに幕
```
